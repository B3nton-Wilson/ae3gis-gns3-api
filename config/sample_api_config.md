- URL: `http://127.0.0.1:8000/scripts/push`
- Method: `POST`
- Payload: 
**To assign IP and start the web server:**
```
{
  "scripts": [
    {
      "node_name": "Server-**01",
      "local_path": "./run_server.sh",
      "remote_path": "/run_server.sh",
      "run_after_upload": true,
      "executable": true,
      "overwrite": true,
      "run_timeout": 10,
      "shell": "sh"
    }
  ],
	"host_override": "http://192.168.56.101",
  "concurrency": 5
}
```
